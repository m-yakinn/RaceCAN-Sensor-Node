# Software

This folder will contain the software tools for RaceCAN Digital Kit.

## Planned Contents

1. Python telemetry simulator
2. Python dashboard
3. CSV data logger
4. Example telemetry logs
5. Setup instructions

## Software Goal

The software will allow users to run a simulated low voltage telemetry system without needing physical hardware.

The simulator will generate CAN style messages, and the dashboard will display live values, active faults, and logged data.
