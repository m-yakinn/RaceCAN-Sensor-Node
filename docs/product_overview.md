# Product Overview

RaceCAN Digital Kit is a downloadable engineering kit for students, robotics teams, motorsports teams, and embedded systems learners.

The kit teaches how a low voltage telemetry system works from the system level down to the software level.

## Target Users

1. Engineering students
2. Formula SAE and Formula Electric teams
3. Robotics teams
4. Embedded systems beginners
5. PCB design learners
6. CAN bus learners

## Problem Being Solved

Many students want to learn vehicle telemetry, CAN communication, firmware structure, and PCB design, but they do not know how all the pieces fit together.

RaceCAN solves this by providing a complete reference project that includes hardware architecture, firmware logic, simulated telemetry, a dashboard, and documentation.

## What the User Gets

The final digital kit will include:

1. System architecture guide
2. CAN protocol guide
3. Fault detection logic guide
4. Firmware template
5. Python telemetry simulator
6. Python dashboard
7. CSV data logger
8. Hardware design notes
9. KiCad PCB files
10. Setup and testing checklist

## Digital Product Advantage

This product can be distributed without physical inventory, shipping, soldering, or manufacturing.

Users can study the files, run the simulator, modify the dashboard, and eventually manufacture their own PCB if they want.
