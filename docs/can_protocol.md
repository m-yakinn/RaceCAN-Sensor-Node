# RaceCAN V1 CAN Protocol

| CAN ID | Message Name | Purpose |
|---|---|---|
| 0x100 | Heartbeat | Confirms that the node is alive |
| 0x101 | Analog Inputs | Sends throttle or brake position |
| 0x102 | Temperature | Sends thermistor temperature |
| 0x103 | Voltage | Sends measured battery voltage |
| 0x104 | Faults | Sends active fault flags |
