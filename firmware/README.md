# Firmware

This folder will contain the firmware portion of RaceCAN Digital Kit.

## Planned Contents

1. Sensor node firmware template
2. Fault detection logic
3. CAN message formatting
4. Simulated firmware logic
5. Embedded system notes

## Firmware Goal

The firmware will model how a low voltage telemetry node reads sensors, checks faults, and sends CAN style messages.

The first version may be simulated in software before being adapted to a physical microcontroller.
